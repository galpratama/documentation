## Cara memakai portal ini

1. Mulai dari halaman [KilatKoding](/kilatkoding) untuk memahami isi boilerplate.
2. Lanjutkan ke [Mulai dengan benar](/kilatkoding/getting-started) dan [Setup lokal](/kilatkoding/local-setup) kalau Anda ingin menjalankan project.
3. Buka [Kustomisasi](/kilatkoding/customization) kalau Anda ingin mengganti branding, pricing, halaman, atau konten.
4. Gunakan halaman teknis seperti [Arsitektur](/kilatkoding/architecture) dan [Database dan storage](/kilatkoding/database-and-storage) saat Anda mulai mengubah kode.

## Tentang KilatKoding

KilatKoding adalah boilerplate SaaS berbasis Next.js untuk developer Indonesia. Repo ini sudah membawa auth Supabase, pembayaran Midtrans dan Doku, email lewat Resend, blog MDX, dashboard, admin panel, AI routes, dan fondasi testing.

Dokumentasi publik ini merefleksikan kondisi kode di repo `kilatkoding-src` per 19 Maret 2026. Jika repo berubah, Anda bisa memperbarui halaman KilatKoding tanpa mengubah struktur portal.

<Tip>
Kalau Anda juga ingin mengubah struktur dokumentasi ini nanti, mulai dari `docs.json`. Mintlify mengatur produk, navigasi, logo, dan navbar dari file itu.
</Tip>
